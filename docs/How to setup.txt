2. Make the Application Run on Startup:
To make your .exe application run on startup, you can add it to the Windows Startup folded r or create a registry entry. Here's how:

Method 1: Add to Windows Startup Folder
Create a Shortcut of the .exe:

Right-click on the .exe file you just created.

Select Create shortcut.

Add Shortcut to Startup Folder:

Press Win + R, type "shell:startup", and press Enter to open the Startup folder.

Copy the shortcut of your .exe file into this folder.

3. Optional

Your antivirus might flag the .exe as suspicious.
To prevent this, add the original folder location of the .exe file (not the shortcut) to your antivirus program’s exception list.
Search YouTube for how to do this based on your antivirus.

4. Optional: Restart the app on every workstation unlock
To ensure EyeGuard.exe always runs fresh each time you log in or unlock your PC (and prevent multiple instances), set up a Task Scheduler entry as follows:

✅ Create the Scheduled Task
Search in Windows search bar "Task Scheduler" or press Win + R, type "taskschd.msc", and press Enter to open Task Scheduler.

In the right panel, click Create Task… (not Basic Task).

    General Tab
        Name: EyeGuard.exe

        Check ✅: Run only when user is logged on

        Check ✅: Run with highest privileges


    Triggers Tab
        Click New…

        Begin the task: On workstation unlock

        Click OK


    Actions Tab
        Click New…

        Action: Start a program

        Program/script: powershell.exe

        Add arguments:
            -Command "Get-Process EyeGuard -ErrorAction SilentlyContinue | ForEach-Object { $_.Kill() }; Start-Sleep -Seconds 2; Start-Process 'C:\Full\Path\To\EyeGuard.exe'"
            ✅ Replace 'C:\Full\Path\To\EyeGuard.exe' with the full absolute path to your .exe file.

            Example: "-Command "Get-Process EyeGuard -ErrorAction SilentlyContinue | ForEach-Object { $_.Kill() }; Start-Process 'C:\Users\Sicaja\Desktop\SmartCodeACADEMY\0.0_Channel_topics\2025.04.23_24_EyeGuard\pythonProject\dist\EyeGuard.exe'"


    Conditions Tab
        Uncheck ⛔: Stop if the computer switches to battery power


    Settings Tab
        Set:
